# AI Website Copy Generator for Local Businesses

## Project Overview
This project demonstrates the use of AI and prompt engineering to generate professional website copy for local businesses.

The website content was created using structured prompts and AI tools.

---

# Business Name
GlowUp Beauty Salon

---

# Objective
To create AI-generated website copy for a local beauty salon including homepage content, services, testimonials, and CTA sections.

---

# Tools Used
- Windsurf Editor
- ChatGPT
- AI Prompt Engineering

---

# Prompt Used

Act as a professional website copywriter.

Create website copy for a local beauty salon named "GlowUp Beauty Salon".

Include:
1. Homepage Hero Section
2. About Us
3. Services Section
4. Why Choose Us
5. Customer Testimonials
6. Contact/Booking CTA

Tone should be:
- Friendly
- Modern
- Professional
- Persuasive

Keep the content website-ready and simple.
